Find Panel

Find Panel
The Find Panel for searching an individual file can be accessed by selecting the "Find..." command from the "Find" menu. In the Find text box, you can specify the text that you are search for. Your search may use either a Simple Regular Expression or an Advanced Regular Expression. See Regular Expressions for more information.

The Replace text box is used to specify the replacement text for the Replace All, Replace, and Replace & Find commands. You can specify if the Replace All command should apply to the entire file or only to the selcted text. Also, with the Ignore Case checkbox, you can specify whether or not a case-sensitive search should be used.
Find Menu

Find Menu
The Find Menu provides access to the Find and Batch Find panels, and to several common find operations. Here is a description of each item:
  • Find: Open the single-file Find Panel.
  • Find Next: Find the next match for the find text specified in the single-file Find Panel.
  • Find Previous: Find the previous match for the find text specified in the single-file Find Panel.
  • Use Selection for Find: Set the find text in the single-file Find Panel to the currently selected text.
  • Scroll to Selection: Scroll the window to the currently selected text in the document.
  • Batch Find: Open the Batch Find Panel.
  • Go To Line Number: Allows you to select a particular line number in a document.
