Purchase Online
Latexian can be purchased through the Taco Software Online Store. After you make a purchase, you will receive an email containing your license code information. You will need to enter your license code into Latexian in order to continue using Latexian beyond the free trial period.
Entering a License Code

Enter License Code
If you purchase Latexian online, you will need to enter your license code into Latexian to use it beyond the trial period. Also, there may be other occasions, like when you get a new computer, that you need to manually enter your license code. To do this, you will need to have the license code email that you received when you purchased Latexian. To enter the license code, select "Enter License Code..." from the "Latexian" menu. Then, fill in the name, email, and license code fields, exactly as they appear in the license code email. To ensure accuracy, it is best if you can copy and paste the values from the email into the form. The license code validation will fail if the values are not entered correctly.
Recovering a Lost License Code
To recover a lost license code, visit the Taco Software Recover License Code page. You will need to provide the email that you used when you ordered Latexian, and your license code will be emailed to you. If you have any problems recovering your license code, contact Taco Software support.
